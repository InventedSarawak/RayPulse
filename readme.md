# Raypulse

### Build Dependencies

- **C++11**
- **OpenGL 4.6**
- **GLFW 3.x**
- **glad**
- **GLM**
- **OpenEXR 3.4**

### Controls

- **ESC** - Exit application
- **S** - Save current frame to EXR (timestamped filename)
